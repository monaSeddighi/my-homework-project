The Basic Certificate

In this online workshop which lasted for three weeks I learned the basics of HTML, CSS and JavaScript. The coding was done on VSCode and I learned how the Developer tool works, how to better communicate with developers, and what development paths are available.The workshop was project based. Each course contained videos with home work that we should submit at the end of each week. At the end of the three-week project we submitted the final project where the SheCode team members checked our work and if the project was valid we had the chance to go to the higher level which was called "The SheCodes Plus".
